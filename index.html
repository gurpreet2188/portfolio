ABOUT This project

My first website written mostly in JS.

This was a intersting but project for as I never built a website primarily in JS, I start the project as usual but with multi pages I though it would be 'fun' to write good part of web site in JS. Home page is written in html and SASS but rest of the page 'Destination', 'Crew' and 'Technology' is written in JS and SASS where all the elements are created in JS and linked with classes from SASS file. The data for JS pages are dynamically loaded from JSON provided with the project along with the FIGMA files.

Writting primarily in JS was challenging in the start as I had to figure out a way to not repeat code for every page so a simple class was created to handle the basic creation of elements and if any page needed anything extra it will be added in their own function. Writting whole 3 pages in single JS was not fun, but the idea of it and the fact that I did it, I'm actually quite happy with the final result.
